## ◆環境
①Windows10

②Visual Studio 2019

## ◆アプリ概要
リクエスト内容により、付箋メモのデータを登録・変更・削除するAPIを作成する。

又、複数ユーザの平行利用を想定し、編集中のユーザ以外は閲覧のみ可能とし、どのユーザが編集しているかもAPIで管理する。

現状は、初期生成プロジェクトである。

## ◆起動コマンド

Visual Studioにて、Ctrl + F5キーでアプリを実行できる。

※デバッグモードでの起動でも可能。

## ◆確認方法

現状、以下のコマンドで疎通確認が取れている。
> curl -D - -X GET "http://localhost:9999/api/TodoItems"
